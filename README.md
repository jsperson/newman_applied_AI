# Newman Applied AI

Coursework repository for Newman Applied AI. The repo is organized by course module, with written responses, experiments, and the final project separated from active module work.

## Repository layout

```text
.
├── module-02-eda-and-prep/
├── module-03-modeling/
├── module-04-advanced-evaluation-feature-engineering/
├── final-project/
├── written-responses/
└── archive/
    ├── drafts/
    └── titanic-experimentation/
```

## Module map

| Area | Repository folder | Focus |
|---|---|---|
| Module 2 | `module-02-eda-and-prep/` | Titanic and churn EDA/preparation notebooks and cleaned datasets. |
| Module 3 | `module-03-modeling/` | Modeling notebooks for Titanic and churn, including refined EDA/prep and multi-model comparisons. |
| Module 4 | `module-04-advanced-evaluation-feature-engineering/` | Advanced evaluation and feature-engineering materials. |
| Final project | `final-project/` | Titanic final project notebook and supporting split/median CSVs. |
| Written responses | `written-responses/` | Learning responses and discussion documents. |
| Archive | `archive/` | Draft/test notebooks and Titanic experimentation copies. |

## Notes

The repository includes datasets used in coursework. Notebooks were authored as course artifacts rather than as a packaged Python project.
